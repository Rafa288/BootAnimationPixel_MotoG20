\# Pixel Boot Animation — Moto G20 (Pixel Experience GSI Only)



A Pixel-style boot animation specifically optimized for the Motorola Moto G20 (720x1600) running Pixel Experience GSI.



⚠️ This module is intended ONLY for Pixel Experience GSIs.

It may not work correctly on stock ROMs or other custom ROMs.



---



\## Features



\- Native resolution: 720x1600 (Moto G20)

\- Designed for Pixel Experience GSI boot sequence

\- Systemless installation via Magisk

\- Safe and fully reversible

\- No boot.img modification required



---



\## Compatibility



Device:

\- Motorola Moto G20 (XT2128-1)



ROM:

\- Pixel Experience GSI (Android 12 / 13 / 14)



Requirements:

\- Unlocked bootloader

\- Magisk 24+

\- Working Pixel Experience GSI installation



---



\## Installation



1\. Download the latest release ZIP.

2\. Open Magisk.

3\. Go to Modules.

4\. Tap "Install from storage".

5\. Select the module ZIP.

6\. Reboot.



---



\## Technical Details



This module replaces:

/system/product/media/bootanimation.zip





It uses Magisk overlay mounting and does NOT modify:



\- boot.img

\- splash/logo partition

\- bootloader



---



\## Warning



This module was built specifically for Pixel Experience GSI.

Using it on other ROMs may:



\- Not change the animation

\- Cause bootanimation issues

\- Be overridden by different partition paths



Flash at your own discretion.
